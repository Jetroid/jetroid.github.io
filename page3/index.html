Bartos Flur

Esoteric Sequencer/Trigger Generator

Bartos Flur was the first synth module I ever programmed, created when I was 17 for my A-level Electronics course.

It's a quirky creature; a sequencer whose step is determined by the input voltage level. It's weird... but fun, too!

Deep Thought

All-In-One Boolean Logic

One day, I was explaining logic gates to a teacher, and joked that I should make a circuit with gates, switches and LEDs that could be used as a learning tool.

That idea inspired Deep Thought, which has all the logic gates plus a section that can act as any gate. Hopefully it won't take 7,500,000 years to come up with answers.

Polygraf

Programmable Drum Sequencer

Polygraf comes from an experiment where I inadvertently created a powerful sequencer that 'normal people' would struggle with. Who knew that only 'nerds' can read binary?

Adding seven-segment displays and a reworked record mode resulted in a drum sequencer that is "like something Roland would have created in 1978".

In development, 2015
Video Credit: Frequency Central

Chronograf

Start/Stoppable Low Frequency Oscillator

Chronograf was designed to be a controller of Polygraf and modules like it; something to tell them when to go, when to stop, and which sequences to play.

My favourite thing about the module is the large variety of waveforms it can produce, which makes it a powerful modulation source.

View at Frequency Central
Photo Credit: Frequency Central
Video Credit: Frequency Central

Seismograf

Drum Sample Module

In Chronograf, we experimened with decaying sine waveforms, which often sound like drums.

Seismograf is a further exploration, replacing the waveform data with samples from some of our favourite electronic drum kits. I created two variations, BD (Bass) and SD (Snare), and hope to create more in the future.

Little Melody

Generative Sequencer

Little Melody is based on a technique that the Director of Frequency Central has used for decades, combining a clock divider with a mixer to create sequences.

This old technique is modernised in Little Melody with spicy features like selectable divison channels, new counting modes, and a quantiser.

View at Frequency Central
Development Prototype, 2019
Video Credit: Frequency Central

High Towers

Programmable Clock Divider

High Towers is a sister module to Little Melody, using the same code but with different hardware.

It's a super flexible clock divider. Each output can be assigned to a different division on the fly. It can count up, down, and various versions of 'up-down'. There is also 'cues' mode, which pulses at intervals of the clock.

Reset Simulation

Boolean Logic Set

Reset Simulation has Deep Thought's logic gate set (but without the user selectable section) in a small form factor.

Legend says that when all the LEDs are lit, the simulation we are living in will be replaced with another, more bizarre simulation. Other sources say this has already happened several times.

Logic Bomb

User Selectable Boolean Logic

Logic Bomb has two independent addressable logic sections, which can act as any logic gate. It evolved from an early prototype of Deep Thought!

A logic bomb is a malicious piece of code that triggers when specific conditions are met. This module probably isn't malicious, though. I think.

Logic Bomb Prototype, 2019
Video Credit: Frequency Central

Tempograf

Start/Stoppable Clock Module

Tempograf is a slimmer version of Chronograf without the advanced features. Chronograf is versatile, but often you end up just using it as a startable/stopable clock source.

In some ways, Tempograf is much more like our original vision for Chronograf before we developed the LFO's feature set.
